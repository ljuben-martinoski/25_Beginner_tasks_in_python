# 25 Beginner Tasks in Python

A collection of 25 fundamental Python programming exercises designed to help beginners master core concepts and programming fundamentals.

## Overview

This project contains 25 beginner-level Python scripts covering essential topics including:
- Basic arithmetic and calculations
- Conditional logic
- String manipulation
- Loops and iterations
- Data structures (lists, tuples, sets, dictionaries)
- Functions and algorithms

## Task List

1. **SumOfTwoNumber.py** - Add two numbers together
2. **OddOrEven.py** - Determine if a number is odd or even
3. **SimpleInterestCalculator.py** - Calculate simple interest
4. **AreaOfaCircle.py** - Compute the area of a circle
5. **LargestOfThreeNumbers.py** - Find the largest among three numbers
6. **PalindromeChecker.py** - Check if a string is a palindrome
7. **StringReversal.py** - Reverse a string
8. **CharacterCounter.py** - Count character occurrences
9. **CharacterCounterAdvanced.py** - Advanced character counting
10. **SubstringFinder.py** - Find substrings in text
11. **WordCounter.py** - Count words in text
12. **MultiplicationTable.py** - Generate multiplication tables
13. **FactorialCalculator.py** - Calculate factorial of a number
14. **PrimeNumberChecker.py** - Identify prime numbers
15. **Fibonaccisequence.py** - Generate Fibonacci sequences
16. **SumOfDigits.py** - Sum the digits of a number
17. **FindTheMaximum.py** - Find maximum value in a collection
18. **ListReversal.py** - Reverse a list
19. **Even_Numbers_Filter.py** - Filter even numbers from a list
20. **CommonElements.py** - Find common elements between collections
21. **TupleConversion.py** - Convert between data types using tuples
22. **WordFrequencyCounter.py** - Count word frequencies
23. **UniqueElements.py** - Extract unique elements
24. **KeyValueSwitch.py** - Swap keys and values in dictionaries
25. **SetOperations.py** - Perform set operations
26. **DictionarySoring.py** - Sort dictionary data

## Getting Started

### Prerequisites
- Python 3.x installed on your system

### Running a Script

```bash
python script_name.py
```

For example:
```bash
python 1.SumOfTwoNumber.py
```

## Learning Path

These tasks are organized to build skills progressively:

- **Tasks 1-5**: Basic arithmetic and number operations
- **Tasks 6-10**: String manipulation techniques
- **Tasks 11-16**: Loops, algorithms, and mathematical operations
- **Tasks 17-26**: Data structures (lists, tuples, sets, dictionaries)

## Tips for Success

1. **Understand the problem** - Read and understand what each script should do
2. **Code along** - Type out the solutions rather than copying
3. **Experiment** - Modify the code and test different inputs
4. **Debug** - Use print statements to understand program flow
5. **Practice** - Repeat tasks until comfortable, then try variations
